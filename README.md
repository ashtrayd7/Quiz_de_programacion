# Quiz_de_literatura

Es un mini juego hecho con Python, esta hecho a base de POO y utiliza algunos conceptos de python

es un divertido juego sobre completar unas frases y esta genial por que tienes vidas entonces si fallas en algunas de las palabras pierdes vidas o si adivimas pues sumas puntos al final del juego puedes verificar tu puntaje, ademas de que tiene una pequeña lista en donde si no te sabes la respuesta la puedes adivinar.

# Pasos para jugar el juego:

-Tener una version de Python instalada
-Abrir python y abrir el archivo de juego
-Tener en una caroeta el archivo py del juego y el archivo "preguntas.csv"
-Ya que tenemos el archivo en python darle play en el icono de reproduccion y listo.
